Excel 驱动

读取每个 Sheet 的 B 列主体名称，结果从 C 列起逐格写入，格式统一。

🌐

多源聚合

同时检索百度新闻、新浪财经、执行信息公开网、企查查，去重后输出。

近一年过滤

🛡️

容错运行

单个数据源失败不影响整体，自动跳过并记录警告，保证全量处理。

工作流程

四步完成舆情扫描

从 Excel 输入到结果回写，全程自动化。

1

读取 Excel

遍历所有 Sheet，提取 B 列每行的主体名称（第 2 行起）。

2

多源并行搜索

对每个主体 × 每个关键词（债券违约 / 商票逾期 / 失信被执行 / 欠税）逐一调用各数据源。

3

过滤 & 去重

按日期过滤近一年数据，按 URL 去重，最多保留 10 条结果。

4

回写 & 保存

每条结果写入对应行 C 列起的单元格，格式：[来源] 日期 | 标题 | URL，保存原文件。

快速开始

三行命令跑起来

Python 3.10+ 环境，安装依赖后即可运行。

Terminal 
# 1. 克隆仓库
git clone https://github.com/gypossible/NegNews-Catching-Agent.git
cd NegNews-Catching-Agent

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行（替换为你的 Excel 文件路径）
python main.py 你的文件.xlsx

# 可选：配置企查查 Cookie
export QCC_COOKIE="从浏览器复制的 Cookie 字符串"
